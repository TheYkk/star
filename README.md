# Github start notifier with Telegram

## Cli flags
```bash
star -h
  -help
        Get Help
  -listen string
        IPv4 address to listen on (default "0.0.0.0")
  -port string
        Port to listen on for HTTP (default "8080")
  -v    Print version

```

## Envrionment veriables

> GITHUB_SECRET

> TELEGRAM_TOKEN

> TELEGRAM_CHAT
